## app/jni ✨

app/jni is contains all dependancies to build DImGui
* See [app/jni/LINUX.md](./LINUX.md) for the full Linux build commands for each dependency.
* See [app/jni/WINDOWS.md](./WINDOWS.md) for the full Windows build commands for each dependency.

All dependencies are included as submodules in `app/jni/`:
* [SDL3](https://www.libsdl.org/)
* [SDL3_image](https://www.libsdl.org/projects/SDL_image/)
* [SDL3_mixer](https://www.libsdl.org/projects/SDL_mixer/)
* [SDL3_ttf](https://www.libsdl.org/projects/SDL_ttf/)
* [ShaderC](https://github.com/google/shaderc)
* [SPIRV-Cross](https://github.com/KhronosGroup/SPIRV-Cross)
* [Assimp](https://github.com/assimp/assimp)
* [cImGui](https://github.com/cimgui/cimgui)
